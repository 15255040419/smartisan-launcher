# 更新日志 (Changelog)

## v1.5.3.2 (2026-04-19) - 视觉与逻辑高度回归

**核心修复与视觉回归**
- **Smartisan 风格弹窗逻辑回归**：彻底废弃原生 Android 样式的更新提示 Toast，重构了 `ApkUpdater` 逻辑。现在“发现新版本”与“已是最新版本”均采用统一的、带高亮圆角按钮的锤子风格对话框。
- **Android 16 布局深度修复**：修正了 `ApplicationProxy` 中针对 Android 16 导航栏（Gesture Bar）高度计算的一个长期偏移错误，彻底消除了桌面 Dock 栏与屏幕底部的异常留白缝隙。
- **上游分支全量对齐**：完整合入了维护作者的所有逻辑补丁，包括编辑模式状态栏动画平滑切换、桌面 16 宫格预览模式的图标坐标计算修正等。

**仓库与更新闭环**
- **更新服务器重定向**：将更新检测 API、主题下载 Host 全面迁移至 `15255040419` 的私有仓库地址，实现完全独立的自动更新生态。
- **CI/CD 发布闭环**：配置了基于 GitHub Actions 的自动化签名打包流程，支持推送标签即自动发布 Release APK。

**性能优化**
- 移除了冗余的调试日志（LOG），减少了检测更新时的 CPU 瞬时占用。
- 优化了更新界面的布局加载速度，解决点击设置时的轻微卡顿感。
