## 更新内容 (v1.5.1-1)

### 🆕 新增功能
- 增加未重绘图标自定义更换功能（点击未重绘图标可进入选择页面）
- 优化未重绘图标列表视觉样式，增加“+”上传引导
- 支持从系统相册选择图片作为自定义图标
- 修复部分已知更新检测问题

### 🛠 修复与改进
- **修复排序崩溃**：重构应用图标排序算法，解决由于访问不存在字段导致的闪退问题。
- **自定义图标修复**：修复了修改未重绘图标后分组打乱的问题，现在严格遵循已重绘/未重绘分组。

### 🎨 视觉优化
- **卡片式布局**：应用图标设置页采用圆角卡片化设计，界面更具质感。
- **动态背景**：列表项根据位置自动应用顶部、中部或底部圆角背景。
- **顶栏修复**：修复标题栏下方半透明阴影导致的视觉重叠问题。
- **文案优化**：精简重绘分组标题文字，右移列表内容使排版更平衡。

### 🚀 核心功能
- **在线更新**：集成 GitHub Release 自动检查更新功能，支持每天自动检测。
- **主题同步**：主题下载地址切换至个人仓库 Release，支持在线下载主题包。
- **自动构建**：配置 GitHub Actions 实现代码提交后自动编译签名。
